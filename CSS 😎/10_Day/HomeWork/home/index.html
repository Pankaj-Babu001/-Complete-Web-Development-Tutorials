Flex-Direction: Row vs Column

Understanding the main axis and cross axis in Flexbox layouts

➡️ Flex-Direction: Row

Item 1
Item 2
Item 3
display: flex; flex-direction: row; /* Default value */
Main Axis: Horizontal (left to right)
Cross Axis: Vertical
Use Case: Navigation bars, button groups, horizontal layouts

⬇️ Flex-Direction: Column

Item 1
Item 2
Item 3
display: flex; flex-direction: column;
Main Axis: Vertical (top to bottom)
Cross Axis: Horizontal
Use Case: Sidebars, card layouts, mobile-first design

🔄 Key Concepts

/* All flex-direction values */ flex-direction: row; /* Default: left to right */ flex-direction: row-reverse; /* Right to left */ flex-direction: column; /* Top to bottom */ flex-direction: column-reverse; /* Bottom to top */ /* Remember: flex-direction affects */ /* - Main axis (primary direction) */ /* - Cross axis (perpendicular direction) */ /* - How justify-content and align-items work */
Pro Tip: When you change flex-direction, the main axis and cross axis switch. This affects how justify-content and align-items behave!

🔧 How Equal-Height Cards Work

The key to equal-height cards is using Flexbox on the container:

.cards-container { display: flex; gap: 2rem; flex-wrap: wrap; } .card { flex: 1; /* Each card takes equal space */ display: flex; flex-direction: column; } .card p { flex-grow: 1; /* Content grows to fill space */ } .card-button { margin-top: auto; /* Button sticks to bottom */ }
Key Points:
• flex: 1 makes each card take equal width
• display: flex on cards creates internal flex layout
• flex-direction: column stacks content vertically
• flex-grow: 1 on paragraphs fills available space
• margin-top: auto pushes buttons to bottom

🚀 Flex-Grow Property

Items expand to fill available space based on their flex-grow values:
Item 1
flex-grow: 1
Item 2
flex-grow: 2
Item 3
flex-grow: 3
.item1 { flex-grow: 1; } /* Takes 1/6 of extra space */ .item2 { flex-grow: 2; } /* Takes 2/6 of extra space */ .item3 { flex-grow: 3; } /* Takes 3/6 of extra space */
How it works: flex-grow determines how much of the remaining space an item should take up. The values are proportional - item 3 will be 3 times larger than item 1 when space is available.

🔍 Flex-Shrink Property

Items shrink when space is limited based on their flex-shrink values:
No Shrink
flex-shrink: 0
Normal
flex-shrink: 1
High Shrink
flex-shrink: 2
.item1 { flex-shrink: 0; } /* Never shrinks */ .item2 { flex-shrink: 1; } /* Default shrinking */ .item3 { flex-shrink: 2; } /* Shrinks twice as much */
How it works: flex-shrink controls how much an item shrinks when there isn't enough space. A value of 0 prevents shrinking, while higher values shrink more aggressively.

💡 Common Patterns

/* Common flex patterns */ .equal-width { flex: 1; } /* Equal width items */ .no-shrink { flex: 0 0 auto; } /* Fixed size, no grow/shrink */ .grow-only { flex: 1 0 auto; } /* Can grow, won't shrink */ .shrink-only { flex: 0 1 auto; } /* Can shrink, won't grow */
