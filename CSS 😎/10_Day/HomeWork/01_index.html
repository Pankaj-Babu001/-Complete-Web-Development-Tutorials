üîë Key Concepts

/* All flex-direction values */ flex-direction: row; /* Default: left to right */ flex-direction: row-reverse; /* Right to left */ flex-direction: column; /* Top to bottom */ flex-direction: column-reverse; /* Bottom to top */ /* Remember: flex-direction affects */ /* - Main axis (primary direction) */ /* - Cross axis (perpendicular direction) */ /* - How justify-content and align-items work */
Pro Tip: When you change flex-direction, the main axis and cross axis switch. This affects how justify-content and align-items behave!
