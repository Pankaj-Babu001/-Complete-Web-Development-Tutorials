2. CSS Units Comparison: px vs rem vs em

Pixels (px)

Absolute units that don't change with parent element size.

This box uses pixels for font-size (16px), padding (16px), and margin (8px).

Root Ems (rem)

Relative to the root element (html) font size.

This box uses rem units for font-size (1rem), padding (1rem), and margin (0.5rem).

Ems (em)

Relative to the parent element's font size.

Parent element with font-size: 20px
This box uses em units for font-size (1em), padding (1em), and margin (0.5em).

4. CSS Inheritance Guide

Property Inherited Description
color Yes Text color is inherited by child elements
font-family Yes Font family is inherited by child elements
font-size Yes Font size is inherited by child elements
line-height Yes Line height is inherited by child elements
text-align Yes Text alignment is inherited by child elements
margin No Margin is not inherited by child elements
padding No Padding is not inherited by child elements
border No Border is not inherited by child elements
background No Background is not inherited by child elements
width No Width is not inherited by child elements
height No Height is not inherited by child elements
display No Display property is not inherited by child elements
