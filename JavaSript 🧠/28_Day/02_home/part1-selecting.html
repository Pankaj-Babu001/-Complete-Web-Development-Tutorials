What is the DOM?

The DOM (Document Object Model) is a programming interface for HTML and XML documents. It represents the page so that programs can change the document structure, style, and content.

Window vs Document

// Window is the global object in the browser console.log(window.innerWidth); // Viewport width console.log(window.location.href); // Current URL // Document is part of the window object and represents the HTML document console.log(document.title); // Page title console.log(document.head); // Head element console.log(document.body); // Body element

// Modern selection methods - more powerful and flexible // querySelector returns the first matching element const firstDiv = document.querySelector('div'); const specialItem = document.querySelector('.special-item'); const dataElement = document.querySelector('[data-category="important"]'); // querySelectorAll returns a NodeList of all matching elements const allDivs = document.querySelectorAll('div'); const allDemoElements = document.querySelectorAll('.demo-element'); // NodeLists are static (don't update automatically) and can be iterated with forEach
