4. Efficient DOM Manipulation with DocumentFragment

DocumentFragment allows us to create a lightweight document structure in memory, make multiple changes to it, and then append it to the DOM all at once. This is more efficient than making individual DOM updates.

// Using DocumentFragment for efficient DOM manipulation
function generateItems() {
    const fragment = document.createDocumentFragment();
    const list = document.getElementById('fragment-list');
    
    // Clear existing items
    list.innerHTML = '';
    
    // Create 100 list items and add to fragment
    for (let i = 1; i <= 100; i++) {
        const item = document.createElement('div');
        item.className = 'fragment-item';
        item.textContent = `Item ${i}`;
        fragment.appendChild(item);
    }
    
    // Append all items at once
    list.appendChild(fragment);
}
