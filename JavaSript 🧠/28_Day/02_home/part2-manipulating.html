Changing Content

Original content with HTML tags and text.
const element = document.getElementById('content-example'); // innerHTML - gets/sets HTML content (parses HTML tags) element.innerHTML = 'New content with HTML tags'; // textContent - gets/sets text content (ignores HTML tags) element.textContent = 'New content with HTML tags'; // innerText - gets/sets visible text (respects CSS, performance issues) element.innerText = 'New content with HTML tags';

Security Warning: XSS Attacks

Using innerHTML with user input can lead to XSS (Cross-Site Scripting) attacks. Always prefer textContent when dealing with user-generated content.

Manipulating Attributes

Element with attributes
const element = document.getElementById('attribute-element'); // Direct property access for standard attributes element.id = 'new-id'; element.className = 'new-class'; // Using getAttribute and setAttribute for any attribute element.setAttribute('data-custom', 'custom-value'); const customValue = element.getAttribute('data-custom'); // Removing attributes element.removeAttribute('data-custom');

ClassList API

Element with classes
const element = document.getElementById('classlist-element'); // classList provides methods to work with classes element.classList.add('new-class'); // Add a class element.classList.remove('base-style'); // Remove a class element.classList.toggle('active'); // Toggle a class element.classList.contains('demo-element'); // Check if has class // Add/remove multiple classes element.classList.add('class1', 'class2', 'class3'); element.classList.remove('class1', 'class2');

Changing Inline Styles

Element to style
const element = document.getElementById('style-element'); // Changing styles directly element.style.backgroundColor = 'red'; element.style.width = '200px'; element.style.height = '100px'; // For CSS properties with hyphens, use camelCase element.style.fontSize = '20px'; element.style.borderRadius = '10px'; // Setting multiple styles at once element.style.cssText = 'color: white; padding: 10px; margin: 5px;'; // Removing a style element.style.backgroundColor = '';
